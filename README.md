# AutoClicker
This is a simple script that will automate mouse clicking
It might be useful in situations where you need to constantly click your mouse and this script can help to prevent you from breaking your mouse
Here is an example to use this in your custom script:
import AutoClicker
''' 
First parameter is the hotkey for starting the clicking
Second parameter is the hotkey for pausing the clicking
Both parameters must be a string and a key on the keyboard
'''
auto_click = AutoClicker.AutoClick('r','t')
